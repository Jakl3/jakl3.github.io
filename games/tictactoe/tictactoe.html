Tic Tac Toe

This project is a recreation of the game TicTacToe, created in the spring of 2020. I used this project as a way to learn graphics outside of 2d-grids, mouse input, and drawing graphics based on that mouse input. Though I stopped working on this a while ago, I am revisiting it to work on implementing a minimax algorithm, so that the computer can make its own logical decisions and ensure that it wins the game. To play, simply click on a square. The code for the project can be found here.
