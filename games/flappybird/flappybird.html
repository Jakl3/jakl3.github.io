Flappy Bird

This project was my introduction into intermediate-level game development using JavaScript, created in 2020 following the COVID lockdown. It is a recreation of the game Flappy Bird, with proper sprites, sounds, and physics. I made this project largely with the help of this tutorial. To play, you can either use the spacebar button or click on the container itself. The code for this project can be found here.
